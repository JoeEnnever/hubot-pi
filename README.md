# hubot-pi

Give digits of pi

See [`src/pi.coffee`](src/pi.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-pi --save`

Then add **hubot-pi** to your `external-scripts.json`:

```json
[
  "hubot-pi"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
